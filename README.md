## Cómo publicarla gratis en GitHub Pages

1. Crear un repositorio nuevo y público.
2. Subir index.html, styles.css y script.js a la raíz.
3. Ir a Settings > Pages.
4. Elegir:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
5. Guardar.
6. Esperar uno o dos minutos.

La URL será similar a:

https://TU-USUARIO.github.io/apoyar-viajemos-con-maxi/

## Cómo actualizar la barra

En index.html buscar:

- 11%
- $11.000
- $100.000
- width: 11%

Cambiar esos valores cuando avance la meta.
